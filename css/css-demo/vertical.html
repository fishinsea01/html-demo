总结下：其实实现垂直居中只有一下几种思路：
1：已知元素高度垂直居中height值与line-height设置为一样即可。
2：采用绝对定位后设置left：50%;top:：50%后,transform: translate(-50%,-50%);补回差距，translate的百分比，是相对于元素自身
3：用css3的flex属性
